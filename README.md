# Course Registration & Mentor Management System

This is a Spring Boot based web application that allows students to register for courses and interact with mentors.

## 🚀 Features

- Student Registration
- Course Enrollment
- Mentor Assignment
- View Registered Courses
- Manage Student Details
- REST API support

## 🛠 Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## 📂 Project Structure

- src/main/java → Application source code
- src/main/resources → Configuration files
- pom.xml → Project dependencies

## 🔗 API Endpoints (Sample)

- GET /students → Get all students
- POST /students → Add new student
- GET /courses → Get all courses
- POST /courses → Add course
- POST /register → Register student to course

## ▶️ How to Run

1. Clone the repository
2. Open in IDE (IntelliJ / Eclipse / Visual Studio Code)
3. Configure MySQL in `application.properties`
4. Run the Spring Boot application
